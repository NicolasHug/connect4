Connect4
========

version number: 0.0.1
author: Nicolas Hug

Overview
--------

A basic connect4 game

Installation / Usage
--------------------

    $ git clone https://github.com/Niourf/connect4.git
    $ python setup.py install

or for development mode:

    $ pip install -e .

Usage
------
    $ python -m connect4

For help

    $ python -m connect4 -h
