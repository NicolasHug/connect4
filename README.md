Connect4
========

version number: 0.0.1
author: Nicolas Hug

Overview
--------

A basic connect4 game

Installation / Usage
--------------------

    $ git clone https://github.com/Niourf/connect4.git
    $ python setup.py install

or for development mode:

    $ pip install -e .


To run a game session:

    $ python -m connect4
